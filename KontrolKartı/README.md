⚙️ KontrolKartı Module - Baggage State Panel

Modern ve modüler yapıda geliştirilen **KontrolKartı Module**, ana proje olan Baggage State Panel yapısının alt modüllerinden biri olarak hazırlanmıştır.

Bu modül; kontrol kartı ayarlarını, sorgu sürelerini, XRAY güç yönetimini, genel sistem ayarlarını ve timeout sürelerini daha düzenli ve hızlı şekilde yönetmek amacıyla geliştirilmiştir.

🚀 Proje Hakkında

Baggage State Panel, bagaj sistemlerinde kullanılan farklı bileşenleri tek panel üzerinden yönetmek amacıyla tasarlanmış modüler bir yapıdır.

Bu repoda yer alan çalışma ise sistemin alt modüllerinden biri olan **KontrolKartı** ekranıdır.

KontrolKartı modülü sayesinde:

- Bilgi alma işlemleri kolayca gerçekleştirilebilir
- Sorgu süreleri ayarlanabilir ve yönetilebilir
- XRAY güç durumu kontrol edilebilir
- Genel sistem ayarlarına hızlıca erişilebilir
- Timeout süreleri düzenlenebilir
- Operasyon süreçleri hızlanır
