# 🗄️ DbManager Module - Baggage State Panel

Modern ve modüler yapıda geliştirilen **DbManager Module**, ana proje olan **Baggage State Panel** yapısının alt modüllerinden biri olarak hazırlanmıştır.

Bu modül; kullanıcı yönetimi, sayaç işlemleri, yetkilendirme kontrolleri ve sistem erişim süreçlerini daha düzenli şekilde yönetmek amacıyla geliştirilmiştir.

---

## 🚀 Proje Hakkında

**Baggage State Panel**, bagaj sistemlerinde kullanılan farklı bileşenleri tek panel üzerinden yönetmek amacıyla tasarlanmış modüler bir yapıdır.

Bu repoda yer alan çalışma ise sistemin alt modüllerinden biri olan **DbManager ekranıdır**.

DbManager modülü sayesinde:

* Kullanıcı işlemleri yönetilebilir
* Yetkilendirme süreçleri kontrol edilir
* Sayaç verilerine erişilir
* Sistem işlemleri daha düzenli hale gelir
* Operasyon süreçleri hızlanır

🧩 Ana Sistem İçerisindeki Yeri

Bu modül aşağıdaki yapı içerisinde yer almaktadır:

* ⚙️ KontrolKartı
* 🔋 UPS
* 🗄️ DbManager ✅
* 📦 Konveyör
* 📝 Logger

Bu projede geliştirilen bölüm: **DbManager**

🗄️ DbManager Modülü Özellikleri
